# Teslo API

1. Levantar las base de datos con docker-compose
```
docker-compose up -d
```
2. Configurar las variables de entorno en el archivo .env.template y renombrar a .env
3. Levantar el proyecto
```
npm start:dev
yarn start:dev
```
